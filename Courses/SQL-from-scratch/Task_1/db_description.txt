Categories : категории товаров с название, картинкой и описанием этих категорий.
Contacts : контакты сотрудников, клиентов и поставщиков;
Customers : информация о клиентах.
Employees : информация о сотрудниках.
Products : информация о товарах.
Suppliers : информация о поставщиках.
Shippers : id грузоотправителя, описание компании и телефонный номер.
Region : информация о регионах.
Territories : id территории его описание и id региона.
Orders : информаци яо заказах.
Order Details : детали заказов.
EmployeeTerritories : id территории к которой прикреплен сотрудник.
